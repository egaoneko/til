# Smooth scroll on mobile

## Style

```css
-webkit-overflow-scrolling: touch;
```

## Compatibility

* Safari: iOS 5
* Android Browser: 3.0
* Blackberry Browser: 6
* Chrome for Mobile
* Firefox Mobile
* IE Mobile: 9
* Opera Mobile
