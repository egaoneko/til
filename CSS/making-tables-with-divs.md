# Making tables with divs

```xml
<div class="table">
    <div class="row">
        <div class="cell"></div>
        <div class="cell"></div>
        <div class="cell"></div>
    </div>
</div>
```

```css
.table { display: table; }
.row { display: table-row; }
.cell { display: table-cell; }
```
