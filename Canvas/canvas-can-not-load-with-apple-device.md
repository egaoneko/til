# Canvas can not load with Apple device

```
Canvas area exceeds the maximum limit (width * height > 16777216).
```

`iOS` 및 `Safari`에서 `canvas`가 보이지 않는 이슈가 발생하였다. 로그를 확인해보니 위와 같이 최대값이 존재하는 것을 확인할 수 있었다.
