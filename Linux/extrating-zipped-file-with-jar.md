# jar로 zip 압축 풀기

### 압축 풀기

```bash
jar xvf zipped_something.zip
```

해당 파일명으로 된 폴더를 만들고, zip 파일을 그 안에 풀어준다.
단, tar나 rar 등은 압축이 풀리지 않는다.
